Proxy Server

Using Java to replicate the increased performance of a Proxy Server. By using Registries and Remote Tasks, the program retrievs website HTML and stores it in a local file. When the same URL is requested, the server checks the local cache first and returns the cached HTML if available. The project demonstrates the efficiency of caching and the benefits of using a Proxy Server for web requests.

Java, Remote Objects, Networking
